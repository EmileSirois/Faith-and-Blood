### Tours de Force

Il est important de noter que toutes les actions et réactions ci-dessous nécessitent l'utilisation d'une charge de **Tour de Force**.

| Nom                       | Arme               | Effet                                                                                                                                                                                                                                             |
| ------------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Twist The Blade           | Dagger             | Après avoir touché avec une attaque, vous pouvez immédiatement effectuer une seconde attaque en tournant la lame dans la plaie.                                                                                                                   |
| Rapid Concision           | Stiletto           | Obtenez l'avantage sur votre prochaine attaque. Si elle touche, ajoutez 2d4 dégâts perforants (_Piercing_).                                                                                                                                       |
| Riposte                   | Rapière            | Après avoir paré une attaque, vous pouvez attaquer immédiatement l'attaquant.                                                                                                                                                                     |
| Defensive Posture         | Épée longue        | Après avoir attaqué, entrez dans une posture défensive jusqu'à votre prochain tour. En posture défensive, les jets de parade sont effectués avec avantage.                                                                                        |
| Lunge                     | Lance              | En attaquant, déplacez-vous de 20 pieds vers votre cible. Si l'attaque touche, elle inflige 1d4 dégâts supplémentaires.                                                                                                                           |
| Bodyshot                  | Dusters            | Infligez une attaque dévastatrice qui inflige le maximum possible sur les dés de dégâts. La cible doit réussir un jet de sauvegarde de Constitution ou être _Dazed_.                                                                              |
| Volley                    | Throwing Knives    | Doublez le nombre de couteaux lancés lors de cette attaque.                                                                                                                                                                                       |
| Off Hand Blow             | Shortsword         | Lorsque vous attaquez avec une _Shortsword_, effectuez immédiatement une attaque avec une arme dans votre seconde main.                                                                                                                           |
| Lacerate, Crush, Puncture | Trench Knife       | En attaquant, ajoutez 1d4 dégâts du type choisi (_Slashing_, _Bludgeoning_ ou _Piercing_). La cible doit réussir un jet de sauvegarde de Constitution ou subir l'effet correspondant (_Bleeding_, _Exposed_ ou _Faltered_).                       |
| Hold the Line             | Halberd            | Lorsqu'une créature entre à portée de mêlée, vous pouvez utiliser votre **Réaction** pour l'attaquer.                                                                                                                                             |
| Shield Bash               | Shield             | Lorsqu'une attaque est **bloquée**, vous pouvez utiliser votre **Réaction** pour attaquer l'assaillant avec votre bouclier, infligeant 1d4 dégâts contondants (_Bludgeoning_). Si l'attaque touche, la cible devient _Exposed_.                   |
| Heavy Hit                 | Falchion           |                                                                                                                                                                                                                                                   |
| Quick Parry               | Parry Dagger       | Lorsque vous échouez un jet de _Parry_, vous pouvez le relancer en y ajoutant votre bonus de maîtrise (_Proficiency Bonus_).                                                                                                                      |
| Unpredicteable weaponry   | improvised Weapons | En attaquant, utilisez votre arme de façon inntendue, infligez _exposed_ à votre cible et doublez les dés d'arme roulés pour le dmg. Cependant, prendre cette action brise votre arme                                                             |
| Armor Piercing Shot       | Heavy Crossbow     | En attaquant, ajoutez +3 au jet d'attaque. Si l'attaque touche, ajoutez 3 dégâts.                                                                                                                                                                 |
| Quick Fire                | Shortbow           | Après avoir attaqué, ajoutez une action **Quick Fire** à vos **AP**. **Quick Fire (Action)** : effectuez une attaque avec le _Shortbow_, puis détruisez cette action après utilisation. Cette action est retirée de vos **AP** à la fin du round. |
| Mighty Shot               | Longbow            | En attaquant, dépensez une action de mouvement. Vous devenez _Exposed_ jusqu'à la fin du round. Attaquez avec avantage et ajoutez 2d4 dégâts si l'attaque touche.                                                                                 |
| Trickshot                 | Revolver           | En attaquant, attaquez avec avantage. Lancez ensuite 1d6 ; sur un résultat de 6, l'attaque devient un coup critique.                                                                                                                              |
| Controlled Breathing      | Rifle              | En attaquant, attaquez avec avantage. Vous pouvez immédiatement _cycle_ le Rifle.                                                                                                                                                                 |
