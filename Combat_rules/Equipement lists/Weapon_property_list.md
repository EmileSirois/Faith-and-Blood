### Propriétées d'armes :

| Nom        | Armes | définition                                                                                                                                                      |
| ---------- | ----- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Light      |       | 2 armes légères peuvent être équippées en même temps                                                                                                            |
| Finnesse   |       | Un magus **peut** utiliser son agilité pour attaquer avec ces armes                                                                                             |
| Thrown     |       | Ces armes peuvent être lancées pour effectuer une attaque à distance                                                                                            |
| Concealed  |       | Ces armes Peuvent facilement être camouflées                                                                                                                    |
| Versatile  |       | Les dégats d'une arme veratile sont augmentés à (n) si utilisée à deux mains                                                                                    |
| Protection |       | L'armure de l'utilisateur est augmenté de (n). Une seule protection s'applique                                                                                  |
| Range      |       | La distance à laquelle l'arme peut être utilisée. La seconde distance cause Désavantage                                                                         |
| Crushing   |       | Les attaques faites contre les ennemis **Faltered** sont faites avec Avantage                                                                                   |
| Heavy      |       | Une arme lourde requiert au moins (n) points de force pour attaquer sans Désavantage                                                                            |
| Two-Handed |       | L'arme prend 2 mains pour utiliser                                                                                                                              |
| Reach      |       | La portée d'attaque est augmentée de 5 pieds                                                                                                                    |
| Reload     |       | L'action de recharger doit être utilisée après (n) attaques                                                                                                     |
| Ammunition |       | Les munitions de cette arme sont particulièrement difficiles à se procurer                                                                                      |
| Cycling    |       | Après chaque atq est mise dans votre **AP** une **AC** cycling. Tant que cette **AC** se trouve dans votre **AP**, vous ne pouvez pas attaquer avec cette arme. |
